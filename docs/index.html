What it does

A glanceable market read, built for the terminal

Everything renders as plain status-line text — it sits quietly under your prompt and refreshes as you work.

$

Live quotes, no API key

Prices pulled from Yahoo Finance's public endpoint, cached so frequent refreshes never hammer it.

▂▅▇

Intraday sparkline

A compact block-character sparkline of the day's movement, right next to the price.

●

Market-status dot

Blinks green while the exchange is open for regular trading, steady red when it's closed.

↻

Rotating symbols

Watch a whole list — stocks, indices like ^GSPC, even crypto like BTC-USD.

⌘

Clickable tickers

Each symbol is an OSC 8 link straight to its Yahoo Finance quote page.

▰

Zero dependencies

Pure Node 18+. Works on Windows, macOS, Linux, and WSL. Tested, single-file core.

Install

Two commands inside Claude Code

Add the marketplace, install the plugin, and the ticker appears in your status line.

# in Claude Code
/plugin marketplace add noam-bash/claude-stock-ticker
/plugin install stock-ticker

Requires Node.js 18+ on your PATH and a Claude Code build with status-line support.

// ~/.claude/stock-ticker.json
{
  "symbols": ["SPY", "NVDA", "AAPL", "TSLA"],
  "rotateSeconds": 10,
  "refreshInterval": 1
}

All keys optional. Add indices (^GSPC) or crypto (BTC-USD) to the list — they rotate in turn.
